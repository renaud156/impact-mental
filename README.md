# Impact Mental - Préparation Mentale Tennis & Sports

Site internet pour coach mental préparateur mental spécialisé tennis et sports de performance.

Ce projet a été généré depuis Figma Make. Le fichier original est disponible sur [Figma](https://www.figma.com/design/kyyJ3kbEipv6KRiOenzdx9/Site-internet-pr%C3%A9parateur-mental).

## 🚀 Démarrage rapide

### Installation

```bash
pnpm install
```

### Développement

```bash
pnpm run dev
```

Ouvre [http://localhost:5173](http://localhost:5173) pour voir le site en local.

### Build pour la production

```bash
pnpm run build
```

## 📁 Structure du projet

```
.
├── src/
│   ├── main.tsx              # Point d'entrée React
│   ├── App.tsx               # Composant principal
│   ├── components/           # Composants réutilisables
│   ├── assets/               # Images, icons, etc.
│   └── styles/               # Styles globaux
├── index.html                # HTML principal
├── vite.config.ts            # Configuration Vite
├── tailwind.config.ts        # Configuration Tailwind CSS
├── postcss.config.mjs        # Configuration PostCSS
└── package.json              # Dépendances
```

## 🎨 Stack technologique

- **React 18** - Bibliothèque UI
- **Vite 6** - Build tool ultrarapide
- **Tailwind CSS v4** - Framework CSS
- **Radix UI** - Composants accessibles
- **React Router 7** - Navigation
- **MUI Icons** - Icônes
